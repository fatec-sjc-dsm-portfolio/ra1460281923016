Jogo da Velha

Curso DIO • 2021

Jogo desenvolvido como parte do curso oferecido pela Digital Innovation One (DIO), focado no desenvolvimento de aplicações interativas com tecnologias web.

×

Sobre o Projeto

Este projeto foi desenvolvido como parte de mais um módulo completo do curso da Digital Innovation One (DIO). O objetivo era construir um clássico jogo da velha, permitindo que duas pessoas joguem presencialmente, no mesmo dispositivo, de forma totalmente offline.

Essa aplicação reforçou a prática de lógica de programação, manipulação de eventos no DOM e controle de fluxo, além de estimular o raciocínio sobre estruturas de dados para verificar condições de vitória ou empate.

Tecnologias Utilizadas

HTML CSS JavaScript

Desafios

O principal desafio deste projeto foi estruturar a lógica que controla as jogadas, garantindo que o sistema identificasse corretamente os momentos de vitória, empate e troca de jogadores.

Além disso, foi necessário desenvolver uma interface clara e intuitiva, que proporcionasse uma boa experiência para os dois jogadores, evitando confusões durante as partidas.

Mesmo sendo um jogo simples, sua implementação exigiu atenção aos detalhes e ao tratamento de exceções, o que foi fundamental para fortalecer minha base em JavaScript e melhorar a organização do código.

Finalizar este projeto me trouxe ainda mais confiança para seguir explorando novas aplicações e desafios no universo do desenvolvimento web.
