🌩️ CloudSpark 🌩️

2º Semestre • 2024 • Equipe CloudSpark

Aplicação para análise de imagens orbitais com detecção automática de nuvens e sombras via Inteligência Artificial.

🗺️ Sobre o Projeto

O CloudSpark visa superar as limitações do sensoriamento remoto óptico em ambientes com alta cobertura de nuvens. A aplicação analisa imagens do sensor WPM do CBERS-4A, detecta automaticamente nuvens e sombras através de uma IA própria e permite o download das imagens processadas.

O sistema oferece suporte tanto para dispositivos móveis quanto para uma interface web responsiva e intuitiva. Todo o processamento é realizado por uma Inteligência Artificial treinada e validada especificamente para o projeto.
