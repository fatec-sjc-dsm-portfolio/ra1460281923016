Sobre o Projeto

No 2° Semestre do curso de Desenvolvimento de Software Multiplataforma da Fatec, 1° Semestre do ano, nosso cliente foi a empresa Jaia Software. Recebemos a proposta de desenvolver um Sistema de Inspeção de Entrada para controle de recebimento de grãos.

Parceiro Acadêmico: Jaia Software

Meu papel no projeto: Development Team

Tecnologias Utilizadas

JavaScript HTML CSS React Express Figma Jira Bootstrap MySQL TypeScript Postman VSCode Git GitHub

Contribuições Pessoais e Desafios

No projeto, atuei de forma completa no desenvolvimento e na organização das funcionalidades, com foco na criação e manutenção de sistemas de usuário e no suporte à integração de diferentes fluxos. Participei da implementação de operações essenciais, como cadastro, autenticação e gestão de informações, além de colaborar ativamente na correção de problemas e no apoio técnico ao time.

Desde o início, o grupo manteve um ritmo consistente, mesmo com o desafio de aprender novas ferramentas e metodologias ao longo do processo. Conseguimos superar as expectativas, entregando um sistema robusto e bem estruturado. Minha atuação foi marcada pela adaptação rápida e pela contribuição efetiva na construção de soluções para diversas demandas do projeto.

Minhas principais competências para a entrega desse sistema, incluíram a habilidade de estruturar e organizar sistemas de gestão de informações, implementar fluxos completos de usuário, prototipar e validar soluções de forma eficiente. Além disso, mantenho uma postura proativa e colaborativa, com forte autonomia, responsabilidade e foco na superação de metas e na qualidade das entregas, sempre visando o sucesso coletivo.
