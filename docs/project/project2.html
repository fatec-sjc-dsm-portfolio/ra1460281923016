Sobre o Projeto

No 2° Semestre do curso de Desenvolvimento de Software Multiplataforma da Fatec, 1° Semestre do ano, nosso cliente foi a empresa Jaia Software. Recebemos a proposta de desenvolver um Sistema de Inspeção de Entrada para controle de recebimento de grãos.

Parceiro Acadêmico: Jaia Software

Meu papel no projeto: Development Team

Tecnologias Utilizadas

JavaScript HTML CSS React Express Figma Jira Bootstrap MySQL TypeScript Postman VSCode Git GitHub

Contribuições Pessoais e Desafios

Como desenvolvedor, fiquei novamente mais focado no backend e no banco de dados, mas sempre ajudava no frontend quando necessário. Neste projeto, apesar dos novos grupos, começamos em um ritmo muito bom, sem dificuldades em relação ao tempo, mesmo tendo que aprender novas ferramentas e tecnologias.

Conseguimos entregar todos os desafios propostos e fomos além das expectativas! Iniciamos o uso de ORM em TypeScript, e eu fui responsável pela parte de usuários: CRUD, login e funcionalidades relacionadas. Também desenvolvi e ajudei no CRUD de outras tabelas como Produto, Pedido e Fornecedor, além de atuar na correção de bugs e suporte em geral ao time.

Hard Skills

Tenho uma forte atuação na área de Backend, onde desenvolvo com muita segurança e qualidade. Tenho também um bom domínio de MySQL, conseguindo estruturar, otimizar e gerenciar bancos de dados de forma eficiente. Já em Frontend, reconheço que ainda estou em fase de evolução, mas sigo estudando para aprimorar minhas habilidades nessa área.

Soft Skills
