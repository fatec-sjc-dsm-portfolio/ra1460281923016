Jogo da Memória

Curso DIO • 2021

Jogo desenvolvido como parte do curso da Digital Innovation One (DIO), explorando conceitos de lógica, manipulação de eventos e interatividade no desenvolvimento web.

×

Sobre o Projeto

Este projeto foi mais um módulo completo realizado durante o curso da Digital Innovation One (DIO). A proposta era desenvolver um jogo da memória, mas decidi torná-lo ainda mais divertido e pessoal, utilizando catálogos de animes como tema para as cartas, o que agregou um elemento cultural e visual bastante atrativo à aplicação.

Além das comparações entre as cartas, implementei efeitos sonoros que reforçam a interatividade do jogo, criando uma experiência mais envolvente para o usuário. Também adicionei pequenas animações para tornar as interações mais dinâmicas e agradáveis.

Tecnologias Utilizadas

HTML CSS JavaScript

Desafios

O principal desafio foi estruturar a lógica de comparação das cartas, garantindo que o sistema identificasse corretamente os pares e bloqueasse as interações quando necessário, evitando cliques excessivos ou simultâneos.

Outro ponto importante foi a implementação dos efeitos sonoros, que exigiu atenção à sincronização dos eventos para garantir que os sons fossem acionados no momento exato das interações.

Além disso, foi necessário realizar uma curadoria das imagens de animes, adaptando o material para manter a estética uniforme e garantir a melhor experiência visual para quem jogasse.
