Sobre o Projeto

No 3° Semestre do curso de Desenvolvimento de Software Multiplataforma da Fatec, 2° Semestre do ano, nosso cliente foi a empresa SIATT. Recebemos a proposta de desenvolver um Sistema de Gerenciamento de Times para Projetos Complexos, com o objetivo de gerenciar estrategicamente os projetos da empresa, tanto de forma macro quanto micro, em um cenário de alta complexidade.

Parceiro Acadêmico: SIATT

Meu papel no projeto: Development Team

Tecnologias Utilizadas

Java JavaScript Bootstrap Jira Docker MySQL Eclipse Postman Spring Boot VSCode Git GitHub Slack React

Contribuições Pessoais e Desafios

No projeto, contribuí amplamente para a estruturação e desenvolvimento do sistema, com foco na implementação e organização de fluxos de usuário, além de participar da criação e manutenção de processos essenciais para o funcionamento do sistema. Fui responsável por iniciar a estrutura do projeto, configurar integrações importantes, implementar autenticação e autorização de usuários, além de colaborar com ajustes e correções que garantiram a estabilidade e eficiência da aplicação.

O grupo passou por mudanças com a entrada de novos integrantes, o que exigiu adaptação e ajustes no ritmo de desenvolvimento. Ainda assim, mantivemos a organização e conseguimos superar todos os desafios, entregando as soluções propostas com qualidade e dentro dos prazos. Minha atuação foi marcada pela capacidade de estruturar processos complexos, adaptar fluxos e apoiar o time com eficiência técnica e organizacional.

Entre minhas principais competências para este desafio, destaco a habilidade de estruturar e integrar sistemas, implementar fluxos completos de autenticação e gestão de usuários, além de realizar testes e garantir a qualidade das entregas. Também mantenho uma postura colaborativa, com forte autonomia, organização e comprometimento, sempre buscando contribuir para o fortalecimento do time e a superação das metas estabelecidas.
