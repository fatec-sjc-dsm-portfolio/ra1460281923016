Sobre o Projeto

No 3° Semestre do curso de Desenvolvimento de Software Multiplataforma da Fatec, 2° Semestre do ano, nosso cliente foi a empresa SIATT. Recebemos a proposta de desenvolver um Sistema de Gerenciamento de Times para Projetos Complexos, com o objetivo de gerenciar estrategicamente os projetos da empresa, tanto de forma macro quanto micro, em um cenário de alta complexidade.

Parceiro Acadêmico: SIATT

Meu papel no projeto: Development Team

Tecnologias Utilizadas

Java JavaScript Bootstrap Jira Docker MySQL Eclipse Postman Spring Boot VSCode Git GitHub Slack React

Contribuições Pessoais e Desafios

Como desenvolvedor, continuei focado majoritariamente no backend e banco de dados, mas agora também participei de algumas tasks no frontend. O grupo se manteve com alguns membros, mas também recebeu novos integrantes durante o semestre, o que impactou um pouco o ritmo de desenvolvimento. Mesmo assim, conseguimos entregar todos os desafios propostos e todas as tasks solucionadas!

Trabalhamos com ORM em Java usando o Spring Boot, começamos a utilizar Docker para o gerenciamento de containers e aprofundamos o uso do Jira para organização de tarefas. Fui responsável por iniciar o projeto backend, configurar a conexão com o banco de dados, alterar rotas, implementar CRUD de usuários, autenticação via JWT, login com inspeção de entrada, autorização de usuários, além de realizar testes, corrigir bugs (principalmente no frontend) e implementar funcionalidades adicionais conforme as tasks.

Hard Skills

Tenho bastante experiência na área de Backend, onde atuo com muita segurança e qualidade. Possuo um bom domínio em MySQL, conseguindo trabalhar de forma eficiente na criação e manutenção de bancos de dados. No Frontend, já evoluí bastante, tenho um conhecimento intermediário que me permite entregar interfaces funcionais e bem organizadas.

Soft Skills

Minhas soft skills sempre foram reconhecidas positivamente pelos times com quem trabalhei. Tenho uma participação ativa nas discussões e projetos, ajudando o grupo a encontrar as melhores soluções. A autonomia é uma das minhas maiores características, consigo tocar projetos com responsabilidade e foco. Também prezo muito pela colaboração, buscando sempre fortalecer o trabalho em equipe. E quando o assunto é entrega e resultado, estou sempre comprometido em superar as expectativas.
