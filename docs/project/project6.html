Sobre o Projeto

O projeto foi desenvolvido para automatizar a extração de informações e a sumarização de grandes volumes de dados textuais oriundos de sistemas de atendimento automatizados, no caso, recebemos dados de planilhas geradas pelo jira e um outro formato, alternativo, próprio da empresa. A solução permite identificar os principais pontos das interações, realizar buscas semânticas e gerar insights diários sobre o atendimento ao cliente.

Nosso sistema integra processamento de linguagem natural, pipelines inteligentes de tratamento de dados e dashboards interativos para visualização dos resultados. A plataforma conta com funcionalidades de classificação automática de atendimentos, sumarização de conversas, busca semântica, além de garantir a segurança e anonimização de dados pessoais.

Meu papel no projeto: Scrum Master e Full Stack e Developer.

Tecnologias Utilizadas

Python TypeScript React React Native PostgreSQL Docker AWS Pinecone Microsoft Presidio LangChain Prisma ORM Jira Figma Next.js Nest.js

Contribuições Pessoais e Desafios

No projeto Apolo, atuei como Scrum Master, liderando a organização e planejamento das atividades, conduzindo as sprints, alinhando os processos e garantindo a colaboração efetiva entre os membros da equipe.

Como desenvolvedor full stack, participei de todas as etapas: da prototipação no Figma, desenvolvimento frontend e backend, até a configuração de banco de dados com PostgreSQL, deploy de ambientes na AWS e integração completa das APIs.

Um dos maiores desafios foi a implementação da busca semântica, utilizando Pinecone como vetor de indexação e LangChain para orquestração. Além disso, atuei na limpeza, tratamento e anonimização dos dados sensíveis com o uso de ferramentas como Microsoft Presidio, garantindo conformidade com a LGPD/GDPR.

O projeto também exigiu forte atuação no desenvolvimento de pipelines de PLN para a extração e sumarização automática de informações, bem como a criação de dashboards intuitivos para apresentação de insights sobre as interações.

Durante todo o processo, desenvolvi competências essenciais como liderança, gestão ágil, engenharia de software, integração de sistemas e segurança de dados, entregando uma solução robusta, escalável e alinhada com as necessidades da empresa parceira.
