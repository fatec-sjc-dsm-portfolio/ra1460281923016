Sobre o Projeto

Projeto desenvolvido na FATEC Prof. Jessen Vidal em parceria com a empresa Tecsus, com o objetivo de criar um sistema completo para o monitoramento ambiental utilizando estações meteorológicas de baixo custo.

Essas estações coletam dados como direção e velocidade do vento, índice pluviométrico, umidade, temperatura e pressão atmosférica, enviando informações periodicamente para o servidor, onde são processadas e exibidas em um portal com dashboards e alertas ambientais.

Parceiro Acadêmico: Tecsus

Meu papel no projeto: Backend Developer e DevOps

Tecnologias Utilizadas

Python JavaScript React MySQL Docker AWS GitHub Actions Arduino IoT Cloud Computing Git

Contribuições Pessoais e Desafios

No projeto, atuei de forma abrangente na construção e integração de sistemas de coleta, processamento e armazenamento de dados, além de contribuir com o desenvolvimento de soluções embarcadas para sensores ambientais. Também fui responsável por estruturar os processos de automação e garantir a qualidade e estabilidade das entregas, promovendo práticas que tornaram o sistema mais robusto e escalável.

Os principais desafios envolveram a integração de múltiplos sistemas, o processamento de dados em tempo real e a adaptação a soluções de monitoramento ambiental. Superamos essas barreiras com uma atuação colaborativa, entregando uma solução eficiente e alinhada às necessidades da empresa parceira. Minha participação foi marcada pela capacidade de estruturar fluxos complexos, integrar sistemas heterogêneos e garantir o bom funcionamento do ambiente de desenvolvimento e produção.

Entre minhas principais competências, destaco a habilidade de estruturar sistemas distribuídos, implementar processos de automação, integrar dispositivos físicos a sistemas de informação e garantir a eficiência e escalabilidade das soluções. Além disso, mantive uma atuação colaborativa, com forte autonomia, organização e capacidade de comunicação, contribuindo para a articulação entre as diversas áreas e assegurando o cumprimento dos prazos e objetivos do projeto.
