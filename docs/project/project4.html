Monitoramento ambiental inteligente utilizando IoT e processamento de dados meteorológicos.

×

Sobre o Projeto

Projeto desenvolvido na FATEC Prof. Jessen Vidal em parceria com a empresa Tecsus, com o objetivo de criar um sistema completo para o monitoramento ambiental utilizando estações meteorológicas de baixo custo.

Essas estações coletam dados como direção e velocidade do vento, índice pluviométrico, umidade, temperatura e pressão atmosférica, enviando informações periodicamente para o servidor, onde são processadas e exibidas em um portal com dashboards e alertas ambientais.

Tecnologias Utilizadas

Python JavaScript React MySQL Docker AWS GitHub Actions Arduino (sistema embarcado)

Minhas Contribuições

  • Responsável pela construção do backend para coleta, processamento e armazenamento dos dados meteorológicos.
  • Apoio no desenvolvimento do sistema embarcado para as estações de sensores (Arduino).
  • Implementação da cultura DevOps no projeto, sendo responsável pelo fluxo de Integração Contínua (CI) utilizando GitHub Actions.
  • Suporte geral ao time em processos de automação e integração dos serviços na nuvem.
